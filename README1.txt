In order to run the code for this project, a user must have WAMP/XAMPP installed and active. MySQL is also recommended.

On PC, open XAMPP and click "Start" on Apache and MySQL. Then, open the file "SystemDesignProjects.php" in browser by right-clicking on the file and opening with a browser or by accessing:

					 "http://localhost/SystemDesignProject/System-Design-Project/SystemDesignProjects.php"
